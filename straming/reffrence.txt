Build a lightweight “Study With Me / Coding Stream Dashboard” web app for OBS Studio integration.

IMPORTANT:
Keep the project SIMPLE, CLEAN, LIGHTWEIGHT, and BEAUTIFUL.
Do NOT overengineer it.
Avoid unnecessary backend complexity.

==================================================
MAIN GOAL
==================================================

Create a premium-looking productivity overlay/dashboard for YouTube coding/study livestreams.

The app will be used inside:
OBS Studio → Browser Source

Purpose:
- Show Pomodoro timer
- Show tasks/goals
- Show productivity stats
- Frame the coding area
- Create a professional stream appearance

The app should feel:
- Minimal
- Calm
- Modern
- Productive
- Easy on the eyes
- Premium
- OBS-friendly

==================================================
IMPORTANT DESIGN DIRECTION
==================================================

DO NOT create:
- overloaded cyberpunk UI
- heavy animations
- cluttered widgets
- too many panels
- flashy gamer aesthetics

Instead create:
- clean productivity dashboard
- minimalist premium design
- modern coding/study aesthetic
- subtle glow effects
- readable typography
- balanced spacing

==================================================
TECH STACK
==================================================

Frontend:
- React
- Vite
- TailwindCSS

Animation:
- Framer Motion (minimal use only)

State:
- React state or Zustand

Persistence:
- LocalStorage only

NO:
- backend
- database
- auth
- microservices
- Electron
- AI features
- Socket.IO

==================================================
OBS REQUIREMENTS
==================================================

The app must:
- run smoothly inside OBS Browser Source
- support transparent/dark background
- use very low CPU
- avoid unnecessary rerenders
- support 1920x1080 layout
- remain readable on livestreams
